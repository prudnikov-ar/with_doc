Пример использования формул:

Когда $D > 0$, уравнение $(ax^2 + bx + c = 0)$ имеет два корня, которые можно найти следующим образом:

$$ x_{1}, x_{2} = {-b \pm \sqrt{D} \over 2a} $$

$$ D = b^2 - 4ac $$
